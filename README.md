Git Bash;
Text-Editor: VsCode;
Betriebssystem: macOS, Windows;

## Installation und npm Befehle

* npm Module installieren
     npm i

* sass in css kompilieren
    - npm run watch:sass

* css Browser-Prefixes generieren
    - npm run prefix:css

* css komprimieren
    - npm run compress:css
